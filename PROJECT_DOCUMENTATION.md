# Project Documentation

## Project Overview
This project is an accessible HTML-only webpage created for the Module 01 Mini Hackathon.  
The theme chosen by the team was **Photography**, a creative hobby enjoyed by many people.  
The webpage includes a clear structure, headings, lists, and descriptive text to maintain accessibility and readability.
